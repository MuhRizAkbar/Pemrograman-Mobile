Proyek ini adalah aplikasi mobile berbasis Android yang dikembangkan menggunakan bahasa pemrograman Kotlin. Aplikasi ini dirancang untuk menyajikan informasi edukatif mengenai sejarah Android serta fitur teknis untuk ekstraksi informasi perangkat secara real-time.





\# Fitur Utama

1\. **Halaman Pertama (Interaktif):** 

&nbsp;  - Menampilkan teks Sejarah Android.

&nbsp;  - Fitur sapaan dinamis berdasarkan input nama pengguna.

&nbsp;  - Manajemen tema (Light/Dark Mode) untuk kenyamanan visual.

2\. \*\***Halaman Kedua (Arsitektur \& Sistem):**\*\*

&nbsp;  - Visualisasi 5 lapisan \*Software Stack\* Android (Application hingga Linux Kernel).

&nbsp;  - Informasi perangkat dinamis meliputi Model, Manufaktur, Versi OS, dan Versi Kernel.

&nbsp;  - Monitoring level baterai dan status konektivitas jaringan (WIFI/Cellular).



\# Detail Teknis

\- **Language		:** Kotlin \[cite: 154]

\- **Min SDK		:** Level 24 (Android 7.0)

\- **API Implementation	:** Memanfaatkan `Build`, `BatteryManager`, dan `ConnectivityManager`. \[cite: 287, 292, 294]
